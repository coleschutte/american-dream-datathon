# American Dream Datathon 2026

## Project Overview

This analysis explores the following question:

**To what extent is the American Dream still attainable in today’s economic environment?**

Using data from the U.S. Bureau of Economic Analysis (BEA), our analysis explores how economic growth translates into improved financial well-being in different states.
